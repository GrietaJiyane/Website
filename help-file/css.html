Sass to Css

If you trying to edit into style, one-page .scss files and changes are not apply.you need some setup for apply changes. If you have any questions that are beyond the scope of this help file, please feel free to Contact me Thanks so much!

We are using very easy to understand and compiles ruby, compass, and sass by command lines.

  • 1- Download and Install Ruby from here:- http://rubyonrails.org/download/
  • 2- After complete installation > open assan html folder > hold Shift + right click
  • 3- type this command line - gem install sass
  • 4- then type this - gem install compass
  • 5- then type this - compass watch
  • Now the compass is watching changes for your .scss file, when you make changes into the .scss file, the compass compile them into .css file
  • Recommended editor - Netbeans IDE 8.01 (easy to use and compile files)


Scss files includes

All .scss files into the scss folder of assan template.

style.scss - for main template

one-page.scss - for one page light version

one-page-dark.scss - for one page dark version

_mixins.scss - for sass mixins like transitions, border-radius, box-shadow, tansform etc.

_variables.scss - for variables like colors fonts etc.


How to change color skin with one change

Open variables.scss file

go to this line -$skincolor - put your color code here
