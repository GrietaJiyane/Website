Overall files structure

This is the files structure for assan template.


Html/
├── Bootstrap/
│   ├── All Bootstrap files includes
├── Css/
│   ├── css files includes
├── js/
    ├── js files includes
├── scss/
    ├── sass files includes like style.scss,_variables.scss, _mixins.scss, one-page.scss
├── cube-portfolio/
    ├── All Cube portfolio files includes
├── font-awesome-4.3.0/
    ├── All font icons files includes
├── img/
    ├── Replace your images with the placeholder
├── mail/
    ├── contact-me.php file include for contact-default page, open it and paste your email into this
├── sky-form/
    ├── sky form files for contact-flat, contact-option, contact advanced, login / register and checkout pages
├── shop-images/
    ├── shop category images includes into this folder
├── rs-plugin/
    ├── Revolution slider files include into this  folder
├── twit-api/
    ├── twitter feeds files
├── Ajax/
    ├── Cube portfolio's ajax files into this folder
